### svg

2D 그래픽을 구현할 때 캔버스처럼 자바스크립트 api로 그래픽을 코딩하는 것이 아닌 html처럼 태그 베이스로 구현할 수 있다.<br>
간단한 차트, 로고, 아이콘 등을 처리할 때 주로 사용된다.
